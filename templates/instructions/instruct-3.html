Instructions

Every trial takes part at a specific location. For example, you might look for treasure in Portugal or in Ethiopia.

You might visit the same locations multiple times. If you return to a location you have already visited you will not be able to check under any more tiles and you will have to remember what you learnt the first time you visited that location.

The treasure does not move, so if you found it before, it will still be under the same tile.
